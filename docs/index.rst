Welcome to STOUT - Smiles-TO-iUpac-Translator's documentation!
==============================================================

.. image:: https://github.com/Kohulan/Smiles-TO-iUpac-Translator/blob/development/docs/_static/STOUT.png?raw=true
  :width: 500
  :align: center

STOUT V2, a transformer-based model that accurately translates SMILES strings into IUPAC names. Trained on a dataset of nearly 1 billion molecules, STOUT V2 achieves high accuracy on test and benchmark datasets. a web application is accessible at stout.decimer.ai.

For comments, bug reports or feature ideas, please use github issues
or send an email to kohulan.rajan@uni-jena.de

Installation
============

Install STOUT in the command line using pip:

.. code-block::

   pip install STOUT-pypi
